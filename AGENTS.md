## Repository Overview
This is my GitHub Profile README repository (`chotushikari/chotushikari`). It contains my public profile landing page with dynamic stats, animations, and project showcases.

## Tech Stack
- Markdown (GitHub Flavored Markdown)
- GitHub Actions (for automation)
- Third-party APIs (GitHub Stats, Streak Stats, Activity Graph, Capsule Render)
- No build step required — pure static Markdown

## Key Files
- `README.md` — Main profile content (THE file)
- `.github/workflows/snake.yml` — GitHub Actions workflow for contribution snake animation
- `AGENTS.md` — This file (repository instructions for AI agents)

- Primary accent color: `#00D4AA` (teal/cyan)
- Secondary accent: `#FF6B6B` (coral red)
- Background: `#0D1117` (GitHub dark)
- Keep it concise — recruiters scan in 6 seconds
- Every project must have: what it does, tech stack badges, and a live demo link
